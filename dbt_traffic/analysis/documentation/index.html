Raw Data Model

Description: This model represents the raw data imported from the pNEUMA dataset.

SQL Code:

SELECT *
FROM source_pNEUMA_data;

Cleaned Data Model

Description: This model performs data cleaning and preprocessing on the raw data.

SQL Code:

WITH cleaned_data AS (
    SELECT
        track_id,
        vehicle_type,
        distance_traveled_meters,
        average_speed_kmh,
        time,
        latitude,
        longitude,
        speed_kmh / 3.6 AS speed_ms,
        longitudinal_acceleration,
        lateral_acceleration
    FROM raw_data
    WHERE -- Add filtering conditions if necessary
)
SELECT *
FROM cleaned_data;
